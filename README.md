#  Social Media Analytics On Excel

A data analytics project that analyzes social media performance across multiple platforms using Excel. The project focuses on data cleaning, engagement analysis, hashtag performance, campaign effectiveness, follower retention, and interactive dashboard creation to generate actionable marketing insights.

---

#  Project Overview

Social media platforms generate large amounts of engagement data that can help businesses optimize their marketing strategies. This project analyzes posts from multiple platforms to evaluate engagement metrics, campaign performance, hashtag effectiveness, and audience retention.

The analysis is performed entirely in Microsoft Excel using Pivot Tables, Pivot Charts, formulas, conditional formatting, and interactive dashboards.
